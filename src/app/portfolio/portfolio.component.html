Maze solving robot

This project was for college level robotics competition held at IIT Kanpur. It is an autonomous robot which can follow line or wall and has the ability to solve a complicated maze. Built with arduino, custom built sensors and electric circuitary. This used PID(Proportional Integral Delivative) control for stabilization, error minimization and course correction.
